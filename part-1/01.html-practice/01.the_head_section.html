Charset or characterset Elements :

In the head section 1 element is character set ot charset :

Computer doesn't understand the character like A,B,C,D,e,f,g computer only understand 0 and 1. this charset move those things into a numeric values.

The first characterset is :

ASCII - (American Standard Code for information interchange) :

it's only represent the characters on english language.

UTF-8 - (Unicode Transformation Format) :

it's represent almost all characters in the world.
